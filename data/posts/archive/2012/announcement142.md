# While we're working hard on polishing version 2.0 for the final release, we should not forget that there have been quite a few important bug fixes in the 1.4.x branch as well. We are thus happy to announce another production quality release in the stable branch: 1.4.2. Version 1.4.2 is the last release in the 1.4.x branch before 2.0. After 2.0 has been released in May, it will become the new stable branch. Stable releases only contain hand-selected bug fixes which were ported back from the development trunk. There are no new features. Notable changes include: - Deadlock fixes - Lots of updates to the rewritten WebDAV interface - Performance improvements when using the Lucene index - Backup to file system produced bad directory structure - Support circular module imports - XQuery processing fixes eXist-db version 1.4.2 is now available on [sourceforge](http://sourceforge.net/projects/exist/files/Stable/1.4.2/).
